This tool is usefull to create a More Windows Forms or WPF style workflow to the Unity UI Toolkit.

It provides a simple way to generate a code file, that takes care of referencing VisualElements for your.
It is doing so based on the name of the element and, when specified, a style that's used as a filter.

You can combine the generated code with your own logic by writing it in a custom partial class,
just like in WPF or Windows Forms.

To open it, just hit Tools / Stonefreak / UITK Code Generator
